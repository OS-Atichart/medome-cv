## ✨ Features

- 🖥️ **VS Code aesthetic** — split editor, file tree, tabs, status bar, minimap
- 🎨 **GitHub Dark theme** — full syntax highlighting with Java token colours
- ⚡ **Single file** — no frameworks, no build step, pure HTML/CSS/JS
- 📱 **Fits on one screen** — designed to show everything without scrolling
- 🔗 **Clickable links** — GitHub & LinkedIn open directly

## 🛠️ Stack

| Layer | Tech |
|---|---|
| Structure | HTML5 |
| Styling | Vanilla CSS (glassmorphism, CSS variables) |
| Logic | Vanilla JS (no dependencies) |
| Font | JetBrains Mono + Inter (Google Fonts) |
| Hosting | GitHub Pages |

## 🚀 Run Locally

No build step needed — just open the file:

```bash
git clone https://github.com/OS-Atichart/medome-cv.git
cd medome-cv
open index.html   # macOS
# or
xdg-open index.html  # Linux
```
